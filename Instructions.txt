Structure


your-startup/
│
├── apps/
│   ├── api/                # Backend (NestJS)
│   ├── web/                # Next.js
│   └── mobile/             # React Native
│
├── packages/
│   ├── ui/                 # Shared UI components (optional)
│   ├── types/              # Shared TypeScript types
│   ├── config/             # Shared configs (eslint, tsconfig)
│   └── utils/              # Shared utilities
│
├── infra/                  # Docker, deployment configs
│   ├── docker/
│   ├── nginx/
│   └── terraform/ (later)
│
├── scripts/                # Dev scripts
│
├── .env
├── docker-compose.yml
├── package.json
└── turbo.json (if using Turborepo)





api/
├── src/
│   ├── modules/
│   │   ├── auth/
│   │   ├── users/
│   │   ├── products/
│   │   ├── inventory/
│   │   ├── cart/
│   │   ├── orders/
│   │   ├── payments/
│   │   └── notifications/
│   │
│   ├── common/
│   │   ├── decorators/
│   │   ├── guards/
│   │   ├── filters/
│   │   └── interceptors/
│   │
│   ├── config/
│   ├── database/
│   ├── jobs/              # background jobs
│   ├── events/            # domain events
│   ├── app.module.ts
│   └── main.ts
│
├── prisma/
│   └── schema.prisma
│
├── test/
└── package.json





web/
├── app/                  # Next.js App Router
│   ├── (auth)/
│   ├── (shop)/
│   ├── admin/
│   └── layout.tsx
│
├── components/
├── features/             # domain-based frontend logic
│   ├── products/
│   ├── cart/
│   └── orders/
│
├── lib/
├── hooks/
├── services/             # API calling layer
└── types/





mobile/
├── src/
│   ├── screens/
│   ├── features/
│   │   ├── auth/
│   │   ├── products/
│   │   ├── cart/
│   │   └── orders/
│   │
│   ├── components/
│   ├── navigation/
│   ├── services/         # API client
│   └── store/            # Zustand/Redux
